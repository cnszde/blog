Hugo - Script für den loaklen Web-Server
08.06.2021 11:55 folder [linux]   label hugo   blog   server   apache  

Für meinen Gedanken und Notizen nutze ich Hugo, ein schlankes, schnelles Framework um Webseiten zu erstellen, ganz ohne php und co. Allerdings möchte ich nicht immer den Hugo eigenen Webserver starten sondern einen Webserver. In meinem Fall ist das Apache. Natürlich funktioniert das ganze auch mit jedem anderen Webserver, wenn man die Pfade anpasst.

Im Hauptverzeichnis des Blogs erstellen wir eine Datei vim transfer.sh mit folgendem inhalt:

1    #!/usr/bin/bash
2    rm -r public/*
3    sudo rm -r /srv/http/*
4    hugo
5    sudo cp -rv public/* /srv/http

Das Script sollte eigentlich selbsterklärend sein.
