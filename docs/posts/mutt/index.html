Mutt
08.06.2021 12:28 folder [Linux]   label email   mutt   muttrc  

Um eMails zu Lesen und Schreiben nutze ich, trotz vieler anderen möglichkeiten immer noch Mutt. Für gMail sieht die .muttrc so aus:

 1  set from = "DIE EMAIL ADRESSE"
 2  set realname = "DER NAME"
 3  set use_from = yes
 4  set imap_user = "DIE EMAIL ADRESSE"
 5  set imap_pass = "DAS PASSWORT"
 6  set folder = "imaps://imap.gmail.com:993"
 7  set spoolfile = "+INBOX"
 8  set postponed ="+[Gmail]/Entwürfe"
 9  set trash = "+[Gmail]/Papierkorb"
10  set record= "+[Gmail]/Gesendet"
11  set smtp_url = "smtp://$imap_user@smtp.gmail.com:465/"
12  set smtp_pass = "DAS SMTP-PASSWORT"
13  ##LOCAL FOLDERS FOR CACHED HEADERS AND CERTIFICATES
14  set header_cache =~/.mutt/cache/headers
15  set message_cachedir =~/.mutt/cache/bodies
16  set certificate_file =~/.mutt/certificates
17  ##SECURING
18  set move = no #Stop asking to "move read messages to mbox"!
19  set imap_keepalive = 900
20  ##Sort by newest conversation first.
21  set sort = reverse-threads
22  set sort_aux = last-date-received
23  ##Set editor to create new email
24  set editor="vim"

Selbstverständlich gibt es noch viel mehr möglichkeiten in der .muttrc aber um schnell mal nur eben Mails abrufen zu können reicht das.
